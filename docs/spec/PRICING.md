# AI 视频生成模型定价说明

本文档说明了 VideoFly 平台 AI 视频生成模型的定价策略。

---

## 模型定价详情

### Sora 2 Lite (OpenAI)

详情页：https://evolink.ai/zh/sora-2

| 时长 | 有水印 | 无水印  |
| ---- | ------ | ------- |
| 10s  | 2 积分 | 3 积分  |
| 15s  | 3 积分 | 4 积分  |

**规格**:

- 支持时长: 10s, 15s
- 宽高比: 16:9, 9:16
- 图片输入: 支持 (可选)
- 默认无水印 (remove_watermark: true)

**Evolink 成本**:
- 有水印: 10s = 1.6 Credits, 15s = 2.6 Credits
- 无水印: 10s = 2.6 Credits, 15s = 3.6 Credits

## 定价策略

- **换算标准**: 1 产品积分 = 1 Evolink Credit
- **价格处理**: 所有 Credits 向上取整到整数
- **利润率**: 按成本定价（1:1），暂不额外加价

---

## 配置文件

定价在以下文件中配置：

1. **用户配置** (`src/config/pricing-user.ts`) - 所有定价参数
2. **计算逻辑** (`src/lib/credit-calculator.ts`) - 积分计算函数
3. **模型配置** (`src/config/credits.ts`) - 后端验证配置
